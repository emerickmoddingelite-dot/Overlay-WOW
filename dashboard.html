Guide d'utilisation
🌐 Hébergement Web (recommandé)
Dépose tous les fichiers dans un dossier sur GitHub Pages, Netlify ou Vercel (plan gratuit).
Ouvre dashboard.html depuis ton domaine et génère tes URLs.
💻 En local (OBS / StreamLabs)
Dépose le dossier sur ton PC. Dans OBS : Source → Navigateur
Colle le chemin local : file:///C:/Overlays/wow_stream/overlay_debut.html?pseudo=Toi…
Résolution : 1920 × 1080 — Coche "Actualiser quand visible"
⚙ Personnalisation avancée
Modifie config.js pour changer les valeurs par défaut sans passer par le dashboard.
Les paramètres URL écrasent toujours la config.
Modifie shared.css pour changer les couleurs globalement via les variables CSS.
⏱ Timer Pause
Ajoute ?brb_secs=300 à l'URL de l'overlay Pause pour régler le timer (en secondes).
Exemple : overlay_pause.html?brb_secs=600 = 10 minutes.
